  Unggah/Upload

  1. Untuk isi buku, kami hanya menerima format pdf dan djvu.
    Untuk cover buku, bisa berupa jpg/jpeg, png, cdr, atau psd.
    Juga menerima compressed file (rar/7z/zip) yang berisikan format file di atas.
  2. File dengan format epub dan mobi harap dikonversi dulu menjadi pdf. Cara konversi format dapat dibaca di Blog PRINT.CO
  3. File dengan format Word (doc/docx) harap dikonversi dulu formatnya ke pdf.
  4. Maksimal ukuran per file adalah 64 MB (enam puluh empat megabytes).
    Jika Anda akan unggah banyak file, pastikan tiap file ukurannya tidak lebih dari 64 MB.
    Aplikasi kami melakukan pengunggahan secara paralel (asynchronous).
    Kami sangat merekomendasikan agar Anda menggunakan browser terbaru di desktop computer/laptop untuk memperlancar proses pengunggahan.
  5. Jika file Anda lebih besar dari 64 MB, gunakan 7-Zip/WinZip/WinRar untuk compress dan split menjadi beberapa parts.
    Bila file lebih dari 1 dan Anda menggunakan komputer dan/atau browser lama, kami sarankan untuk kompres file-file cetak dalam satu bundle zip / rar / 7z sebelum unggah ke sistem kami.
    Atau unggah file ke online storage Anda, lalu copy-paste tautan file di fitur Tautan Luar.

  Tautan Luar

  1. Untuk isi buku, kami hanya menerima format pdf dan djvu.
    Untuk cover buku, bisa berupa jpg/jpeg, png, cdr, atau psd.
    Juga menerima compressed file (rar/7z/zip) yang berisikan format file di atas.
  2. File dengan format epub dan mobi harap dikonversi dulu menjadi pdf. Cara konversi format dapat dibaca di Blog PRINT.CO
  3. File dengan format Word (doc/docx) harap dikonversi dulu formatnya ke pdf.
  4. Mohon untuk mengatur privasi akses file tersebut agar dapat diakses oleh publik (publicly accessible) agar tim kami dapat mengunduh file cetak Anda.
  5. Gunakan , (koma) sebagai pemisah tautan. Boleh digabung dengan enter/return (baris baru).
    Contoh:
    https://www.dropbox.com/s/.../fileAnda,
    https://drive.google.com/file/d/.../fileBersangkutan,
    https://onedrive.live.com/?authkey=....fileLainLain
    Jangan lupa untuk menghapus tanda koma di belakang URL tautan terakhir.
  6. Tombol UPDATE akan di-enable apabila tautan memiliki http atau https sebagai protokol tautan dan aplikasi kami akan periksa (setidaknya) beberapa hal untuk validitas tautan file Anda.
  7. Untuk duplicated links, kami hanya ambil satu saja.
  8. Setelah selesai paste teks tautan, jika semua tautan sudah benar, klik UPDATE untuk buffer tautan file di browser sebelum masuk ke dalam sistem kami.

 PILIHAN JASA KURIR

Form pemesanan online PRINT.CO menyediakan beberapa pilihan untuk kurir:

  • Pengiriman mencakup hampir seluruh pelosok Indonesia, dan Anda dapat cek pengiriman secara online di PRINT.CO.

    Perhitungan biaya adalah:
    (BIAYA KIRIM) = (BERAT PAKET) × (BIAYA / KG)*
    *(BIAYA / KG) untuk tujuan pengiriman dari BANDUNG (KOTA)

    Kini tersedia pilihan tipe OKE, REG, dan YES (Yakin Esok Sampai).

    Bedanya tipe OKE dan REG: untuk JNE tipe OKE, biaya per kilogram lebih murah daripada JNE REG. Namun, waktu sampai lebih lama* dibanding tipe REG.
    *Tergantung pihak JNE

    Untuk tipe YES, cakupan wilayah kirim tidak seluas OKE dan REGULAR (JNE OKE & REG). Namun, Anda bisa yakin bahwa paket akan tiba keesokan harinya (setelah order selesai dikerjakan).

    Service CTC adalah CITY COURIER, yang berarti tujuan pengiriman berada di satu wilayah dengan asal pengiriman. Untuk CTC saja artinya setara dengan REG, CTCOKE setara dengan OKE, dan CTCYES setara dengan YES.

    Jika tujuan kirim tidak tercakup oleh JNE, Anda dapat memilih beberapa opsi kurir lainnya di bawah ini.

  • Pengiriman mencakup seluruh pelosok Indonesia, dan Anda dapat cek pengiriman secara online di PRINT.CO.

    Perhitungan biaya juga sama seperti JNE, per kilogram (per 1000 gram).

    Biaya bisa lebih murah atau lebih mahal (atau lebih cepat / lama sampainya) daripada jasa kurir lainnya bergantung tujuan dan tipe service.

  • Pengiriman mencakup hampir seluruh pelosok Indonesia. Pengiriman dapat dilacak secara online, namun Anda tidak dapat cek di PRINT.CO. Kami sediakan tautan untuk cek via AfterShip.

    Perhitungan biaya juga sama seperti JNE, per kilogram (per 1000 gram).

    Biaya bisa lebih murah atau lebih mahal (atau lebih cepat / lama sampainya) daripada jasa kurir lainnya bergantung tujuan dan tipe service.

  • Pengiriman mencakup hampir seluruh pelosok Indonesia, dan Anda dapat cek pengiriman secara online di PRINT.CO.

    Perhitungan biaya juga sama seperti JNE, per kilogram (per 1000 gram).

    Biaya bisa lebih murah atau lebih mahal (atau lebih cepat / lama sampainya) daripada jasa kurir lainnya bergantung tujuan dan tipe service.

WARNA

Ketentuan pembulatan jumlah halaman untuk WARNA adalah sebagai berikut:

  • Untuk ukuran A4 dan B5, jumlah halaman buku dibulatkan ke atas (round up) menjadi kelipatan 4.
    Contoh:
    1. Buku dengan 23 halaman 24 halaman (round up ke kelipatan 4).
    2. Buku dengan 132 halaman tetap 132 halaman (sudah kelipatan 4).
    3. Buku dengan 210 halaman 212 halaman (round up ke kelipatan 4).
